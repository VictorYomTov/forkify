STEPS TO RUN:

0. node.js required so download from the official site
1. Go to https://www.food2fork.com/ and sign up.
2. Login --> Browse ---> Recipe api ---> copy your api key.
3. In forkify folder go to src and then to js folder and open "config.js".
4. Change the key value to your api key.
5. Open the cmd on forkify folder and write npm install
6. After you wrote npm install it will install in your computer all the staff the aplication need , wait until it finish
7. Finally write in cmd "npm run start".
